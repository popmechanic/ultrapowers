**Acceptance:** suite — ultrapowers' own skill/test/eval-tooling development; author and operator read the diffs; the committed suite plus the surviving drift pins are the verification.

**Goal:** Ship the 0.1.0 subtraction release (delete prose-pin/self-policing tests and compiler diagnostic ballast, demote sealed acceptance to opt-in, freeze the verification periphery) plus the minimal A/B eval kit that gates the tag, per `docs/superpowers/specs/2026-07-10-0.1.0-subtraction-and-ab-eval-design.md`.

**Architecture:** Seven independent implementation tasks — three build the eval kit under `evals/` (webapp fixture, runner, judge), four subtract (standalone deletions, rule-trims, compiler trim, sealed demotion + freeze). Nothing here touches `waves.js`, the gate scripts, or the trust checks. The eval runs and the tag itself are post-merge runbook items.

**Tech Stack:** Python 3.11 + pytest (kit + suite), bash, Node ≥20 built-ins only (webapp fixture: `node:http`, `node:test`), `claude -p` (runbook only, never CI).

## Global Constraints

- No `anthropic` SDK and no `ANTHROPIC_API_KEY` anywhere — LLM work rides `claude -p` (CLAUDE.md rule). No new pip/npm dependencies.
- The suite (`python3 -m pytest`) must be green at the end of every task; deletion tasks may only delete/trim — never re-add deleted machinery to appease a kept test (if a kept test fails, the trim missed a pin: trim it).
- No test may invoke the `claude` binary — eval-kit pytest coverage exercises dry-run/assembly paths only.
- Fixture layout contract (consumed by the runner): `evals/fixtures/<name>/plan.md`, `evals/fixtures/<name>/project/` (starter repo content), optional `evals/fixtures/<name>/acceptance/<seal-id>/` (sealed-exam vault entries to install).
- Results layout contract (produced by the runner, consumed by the judge): one JSONL row per run appended to `evals/results/runs.jsonl` with keys `{fixture, engine, engineRef, startedAt, wallClockSec, outputTokens, gateVerdict, redirectRounds, falseBlocks, rerunOf}`; the integrated diff saved to `evals/results/diffs/<fixture>-<engine>.diff`.
- Do not modify: `skills/ultrapowers/harnesses/waves.js`, `gate_check.py`, `ultra_gate.py`, `run_lock.sh`, `run_acceptance.sh`, `collect_seal.py`, `seal_hash.py`, `agents/seal-author.md` (the freeze covers them; sealed stays functional opt-in).

```python
#!/usr/bin/env python3
"""A/B eval runner — one cell per invocation, run serially by hand.

Adapted from the removed dev eval harness (git 589114e^): local-marketplace
engine pinning, headless claude -p drive, transcript token harvest, fixture
seal installs. Protocol: docs/superpowers/specs/2026-07-10-*-design.md Part 2.
Never runs in CI; tests cover assembly/harvest only.
"""
import argparse, json, shutil, subprocess, sys, tempfile, time
from datetime import datetime, timezone
from pathlib import Path

RESULTS = "evals/results"


def build_run_plan(engine_ref, engine_label, fixture, root):
    fdir = Path(root) / "evals/fixtures" / fixture
    if not (fdir / "plan.md").is_file():
        sys.exit(f"unknown fixture: {fixture}")
    seals = sorted(p.name for p in (fdir / "acceptance").iterdir()
                   if p.is_dir()) if (fdir / "acceptance").is_dir() else []
    return {
        "fixture": fixture, "engine": engine_label, "engineRef": engine_ref,
        "planPath": str(fdir / "plan.md"),
        "projectDir": str(fdir / "project"),
        "diffPath": str(Path(root) / RESULTS / "diffs" / f"{fixture}-{engine_label}.diff"),
        "rowsPath": str(Path(root) / RESULTS / "runs.jsonl"),
        "sealInstalls": seals,
    }


def harvest_row(transcript_path, started_at, wall_clock_sec):
    tokens = 0
    for line in Path(transcript_path).read_text().splitlines():
        try:
            evt = json.loads(line)
        except json.JSONDecodeError:
            continue
        tokens += (evt.get("usage") or {}).get("output_tokens", 0)
    return {"startedAt": started_at, "wallClockSec": wall_clock_sec,
            "outputTokens": tokens, "rerunOf": None}

# install_seals(plan, root)                                 (seal_fixture.py)
# clone_project(plan) -> workdir                            (git init + commit)
# drive_run(workdir, plan) -> (transcript_path, gate)       (night_runner.py)
# save_diff(workdir, plan)  — git diff base...integration > diffPath
# collect_counters(gate_report) -> {gateVerdict, redirectRounds, falseBlocks}


def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--engine-ref", required=True)
    ap.add_argument("--engine-label", required=True, choices=["A", "B"])
    ap.add_argument("--fixture", required=True)
    ap.add_argument("--dry-run", action="store_true")
    ap.add_argument("--rerun-of", default=None)
    args = ap.parse_args()
    root = Path(__file__).resolve().parents[1]
    plan = build_run_plan(args.engine_ref, args.engine_label, args.fixture, root)
    if args.dry_run:
        print(json.dumps(plan, indent=2)); return
    started = datetime.now(timezone.utc).isoformat()
    t0 = time.monotonic()
    # prepare_engine → install_seals → clone_project → drive_run → save_diff
    ...
    row = harvest_row(transcript, started, round(time.monotonic() - t0, 1))
    row.update({"fixture": plan["fixture"], "engine": plan["engine"],
                "engineRef": plan["engineRef"], "rerunOf": args.rerun_of,
                **collect_counters(gate_report)})
    with open(plan["rowsPath"], "a") as f:
        f.write(json.dumps(row) + "\n")
    print(json.dumps(row, indent=2))


if __name__ == "__main__":
    main()
```
